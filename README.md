# 📱 Radio Survey App

## 🚀 Overview

The Radio Survey App is an interactive Android application that collects user information through a multi-step form using RadioButtons. It ensures smooth navigation, validation at each step, and stores submission history locally.

---

## ✨ Features

* 🧾 Multi-step form with progress tracking
* 🔘 RadioButton-based selections for structured input
* ✅ Input validation at every step
* 💾 Stores submission history using SharedPreferences
* 📜 View and clear previous submissions
* 🔄 Reset form and start again

---

## 🛠 Tech Stack

* Java
* XML
* Android Studio
* SharedPreferences
* Material Design Components

## ⚙️ How It Works

1. User enters name
2. Navigates through multiple steps
3. Selects options using RadioButtons
4. Submits form
5. Data is saved with timestamp
6. User can view or clear submission history

---

## ⚙️ How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Device
